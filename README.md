# Heart Disease Exploratory Data Analysis

This repository contains an EDA on the Heart Disease Prediction dataset on Kaggle by Neurocipher.

The purpose is to uncover patterns, correlations, and key findings that assists in guiding future predictive modeling. 

**Dataset:** [Heart Disease dataset on Kaggle by Neurocipher](https://www.kaggle.com/datasets/neurocipher/heartdisease/data)
